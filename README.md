# 🏥 BMI Calculator (Java Swing)

A feature-rich **Body Mass Index (BMI) Calculator** built with **Java Swing**. This desktop application calculates your BMI, categorises your result with colour-coded feedback, and — via the built-in **AI Health Plan** feature — generates a fully personalised health plan using the **Claude API (Anthropic)** and delivers it directly to your email inbox.

---

## 🚀 Features

- Clean and responsive GUI using Java Swing
- Supports both unit systems:
  - **Metric** — kg, meters
  - **Imperial** — lbs, inches
- Instant BMI calculation with colour-coded result display
- BMI categories: Underweight · Normal weight · Overweight · Obese · Severely obese
- **"More Info" link** — appears after calculation to trigger the health plan flow
- **AI-powered personalised health plan** generated by Claude (Anthropic API)
- **Automated email delivery** via Gmail SMTP — plan sent directly to the user's inbox
- Rich HTML email with plain-text fallback
- Input validation and error handling throughout

## ▶️ How to Run

### 1. Clone or download the project

```bash
git clone https://github.com/your-username/bmi-calculator.git
cd bmi-calculator
```

### 2. Configure your credentials

Open `BmiCalculator.java` and update the following in the `sendEmail()` method:

```java
final String FROM_EMAIL   = "your-email@gmail.com";
final String APP_PASSWORD = "xxxx xxxx xxxx xxxx";
```

### 3. Compile

**Windows:**
```bash
javac -cp ".;javax.mail.jar" BmiCalculator.java
```

**macOS / Linux:**
```bash
javac -cp ".:javax.mail.jar" BmiCalculator.java
```

### 4. Run

**Windows:**
```bash
javac -cp ".;*" BmiCalculator.java
java -cp ".;*" BmiCalculator
```

**macOS / Linux:**
```bash
java -cp ".:javax.mail.jar" BmiCalculator
```

> ✅ Make sure `javax.mail.jar` is in the **same folder** as `BmiCalculator.java`, or provide the full path to the jar file.

## 🤖 AI Health Plan — How It Works

After calculating their BMI, the user clicks **"More Info"**. The following flow is triggered:

The prompt instructs to write a structured plan covering:

1. Friendly greeting and BMI explanation
2. Current health status and associated risks/benefits
3. Diet plan — foods, timing, hydration
4. Exercise plan — type, frequency, duration
5. Lifestyle habits — sleep, stress, daily routines
6. Mental health tips — body image, motivation
7. Motivational closing message

## 📧 Gmail Setup — App Password

To enable email sending, generate a **Gmail App Password**:

1. Go to your Google Account → **Security**
2. Enable **2-Step Verification**
3. Go back to Security → **App passwords**
4. Select app: **Mail** | device: **Windows Computer**
5. Copy the 16-character password
6. Paste it into `APP_PASSWORD` in `BmiCalculator.java`

**SMTP settings used by the app:**

| Setting | Value |
|---|---|
| Host | `smtp.gmail.com` |
| Port | `587` |
| Auth | `true` |
| STARTTLS | `true` |

## ⚠️ Error Handling

| Scenario | Behaviour |
|---|---|
| Non-numeric input | Dialog: "Invalid input!" |

| Invalid or empty email | Dialog: "Please enter a valid email address." |
| Claude API failure | Status: "Error generating plan. Please try again." |
